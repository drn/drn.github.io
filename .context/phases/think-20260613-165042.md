## Handoff (for /dev)

### Key decisions

- Replace the React `<input>` + `Row` rendering with **xterm.js** (`@xterm/xterm`); keep the `Executor`/`Runnable`/`Nameable` registry but change `Runnable.run()` to return `result?: string` (ANSI) instead of `ReactNode`.
- Rich output (`whoami`/`help`): **Nerd Font glyphs** ( / ) + **clickable URLs** via `WebLinksAddon`. Drop `@fortawesome/*` and `react-anime`.
- Output animation: **typewriter** char streaming (replaces fade).
- Status bar: drawn **inside xterm** as a protected **top** row via DECSTBM scroll region (`\x1b[2;<rows>r`), redrawn on a 1s interval and on resize.
- Failure conveyed by **red output**, not by recoloring typed input.
- `Executor` constructor takes `clear: () => void` (was `setContents`).

### Implementation phases

1. **Foundations** — deps, `ansi.ts`, `Runnable` signature, `Executor` clear-callback + test.
2. **Parallel build** — command ANSI conversions; pure `LineBuffer` (+ test); `statusBar` port.
3. **Integration** — `Shell.tsx` host, index/css rewire, mocked-xterm `Shell.test.tsx`, delete `Row`/`Header`/`Input`/`useWindowDimensions`.
4. **Gate** — `yarn install` → `tsc` → `test` until green.

### New files

`src/Terminal/ansi.ts`, `src/Terminal/LineBuffer.ts` (+ `.test.ts`), `src/Terminal/statusBar.ts`, `src/Terminal/Shell.tsx` (+ `Shell.test.tsx`).

### Deleted files

`src/Terminal/{Row,Row.test,Header,Input,Input.test}.tsx`, `Header.css`, `Input.css`, `src/hooks/useWindowDimensions.tsx`.

### Open questions

- Final typewriter per-char delay (snappy vs dramatic) — tune in manual verification.
- Column breakpoints converted from px (~90 / ~150 cols) — verify on resize.
- Confirm `yarn install` is permitted in the sandbox (network).

### Risks

- xterm in jsdom → mock `@xterm/xterm` in `Shell.test.tsx`; keep edit logic in pure `LineBuffer`.
- Font metrics before load → `document.fonts.load('15px "Menlo Nerd Font"')` before `new Terminal`.
- DECSTBM cursor quirks → DECSC/DECRC save/restore around row-1 redraw; re-arm region on resize.
